SMK GUNSANAD PORTAL - Versi 2.0 (Dwibahasa)
==========================================

Bahasa Melayu (BM):
-------------------
Portal ini termasuk Sistem Jual Beli Jadual Waktu yang menggunakan Google Sheets sebagai pangkalan data (Apps Script Web App).
Untuk menggunakan:
1. Pastikan Apps Script yang mengendalikan Google Sheet telah dideploy (Web app URL).
2. Jika anda gunakan web app URL lain, edit jual-beli.html dan gantikan const SCRIPT_URL dengan URL baru.
3. Upload folder 'smkgunsanad-portal-v2' ke GitHub repo dan aktifkan GitHub Pages (main branch / root).

English:
--------
This portal includes the Teaching Schedule Trade System which uses Google Sheets as a backend (Apps Script Web App).
To use:
1. Ensure your Apps Script Web App is deployed and set to allow access.
2. If using a different URL, edit jual-beli.html and replace const SCRIPT_URL with your web app URL.
3. Upload the 'smkgunsanad-portal-v2' folder to a GitHub repo and enable GitHub Pages (main branch / root).
